Cyber Security Password Manager is a Java application designed to enhance password security and user awareness. The project provides features such as user registration, secure login, random password generation, and password strength analysis. It demonstrates core Java programming concepts including Object-Oriented Programming (OOP), file handling, regular expressions, and basic cybersecurity principles.

Key Features:
• User Registration and Login
• Secure Password Generator
• Password Strength Checker
• Menu-Driven Interface
• Cyber Security Best Practices

Technologies Used:
• Java
• OOP Concepts
• Regular Expressions
• Random Class
• Scanner Class

This project was developed as part of my learning journey in Java programming and Cyber Security at Sathyabama Institute of Science and Technology.
